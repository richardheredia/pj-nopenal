# Prototipo del Portal del Poder Judicial del Perú

Versión HTML, CSS y JavaScript con Bootstrap 5, pensada para publicación en GitHub Pages.

## Contenido
- `index.html`: portada institucional mejorada
- `servicios.html`: página interna de ejemplo
- `partials/header.html`: cabecera reutilizable
- `partials/footer.html`: pie reutilizable
- `css/header.css`: estilos de la cabecera y navegación
- `css/body.css`: estilos del contenido principal
- `css/footer.css`: estilos del pie
- `js/includes.js`: carga de parciales HTML

## Características
- Diseño responsive para escritorio, tableta y móvil.
- Menú superior con colapso en pantallas pequeñas.
- Banner principal, accesos rápidos, servicios, noticias y bloque CTA.
- Estructura simple para crecer hacia más páginas internas.
